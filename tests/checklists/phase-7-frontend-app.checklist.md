# Phase 7 Acceptance Checklist — Frontend App

## Navigation & Access
- [ ] Публичные и приватные маршруты разделены.
- [ ] На приватных роутингах без сессии происходит редирект на login.

## Progress & Locking
- [ ] UI корректно отображает статусы locked/available/completed.
- [ ] Locked-уроки не открываются через UI.
- [ ] После completion/pass UI синхронизируется с серверным прогрессом.

## AI Modes UX
- [ ] Lecture отображает потоковый ответ и действие “Завершить урок”.
- [ ] Exam отображает фиксированный набор вопросов с итогом.
- [ ] Consultant доступен только при выполнении условия по модулю.

## Resilience
- [ ] Reconnect не дублирует сообщения.
- [ ] Ошибки лимитов и сети имеют отдельные понятные состояния.
- [ ] На мобиле доступно переключение “Материал/Чат”.
