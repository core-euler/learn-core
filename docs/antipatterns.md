# Antipatterns

## [AP-001] bcrypt/passlib incompatibility in local env

**Статус:** Решено

**Симптом:**
Тесты auth регистрации падали при хэшировании пароля с ошибкой backend bcrypt.

**Причина:**
Локальная комбинация версий `passlib` и `bcrypt` давала runtime-конфликт.

**Решение:**
Во временном bootstrap-слое заменён механизм хэширования на детерминированный salted SHA-256, чтобы не блокировать TDD цикл.

**Примечание:**
Для production-итерации вернуться к устойчивой password KDF политике и зафиксировать версии зависимостей.
