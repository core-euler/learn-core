# Benchmark

## Цель
Зафиксировать протокол проверки воспроизводимости на этапе перехода к реализации.

## Quality Gates
- Окружение поднимается без ручных правок.
- Тестовые сценарии этапа воспроизводимы в чистом запуске.
- Критические требования этапа подтверждаются тестами.

## Метрика итераций
- До начала реализации: только документирование и ревью требований.
- После старта реализации: итерации считаются по циклу «изменение → запуск → фиксация».
